# 퍼니(Funni) - 스튜디오 대관·예약 O2O 플랫폼

## 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 클라이언트 | 퍼니 |
| 서비스 | 촬영 스튜디오 대관·예약 O2O 플랫폼 |
| 예산 | ₩42,700,000 (VAT 별도) + 추가 옵션 ₩3,700,000 |
| 기간 | 16주 (4개월) |
| 기술 스택 | React Native (iOS+Android), React (Web Admin), 토스페이먼츠 |
| 레퍼런스 | 바비톡 (web.babitalk.com) |

## 프로토타입

**URL:** https://funni-prototype.vercel.app

| 화면 | 경로 | 설명 |
|---|---|---|
| 메인 | `/` | 프로토타입 선택 + 앱 구조 안내 |
| 소비자 화면 | `/consumer` | 스튜디오 탐색 → 예약 → 결제 플로우 |
| 업체 화면 | `/business` | 대시보드 → 예약 관리 → 정산 |
| 어드민 웹 | `/admin` | 업체 관리 → 수동 정산 → 광고 |

## 확정 사항

- [x] PG사: 토스페이먼츠
- [x] 정산: 관리자 수동 정산 (자동 정산 없음)
- [x] 수수료율: 어드민에서 운영자가 직접 수정 가능
- [x] 업체 다중 카테고리 등록 가능, 카테고리별 가격·설명 별도 설정
- [x] 업체당 사진 최대 30장, 동영상 불가
- [x] 소셜 로그인: 카카오 + 네이버 (유저당 각 1개씩 총 2개 연동)
- [x] 접속 채널: 어드민 → 웹 / 고객·업체 → 앱 (하나의 앱)
- [x] 통합 네비게이션: 홈 / 탐색 / 예약 / MY (4탭 고정, 역할별 내용 분기)
